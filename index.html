Sobre o Evento

O grupo de usuários de Software Livre Tchelinux, em parceria com o curso Análise e Desenvolvimento de Sistemas, da Faculdade Anglicana de Erechim, tem o prazer de convidar a comunidade para participar do evento que ocorrerá no dia 30 de setembro de 2017 a partir das 8:30h na FAE em Erechim.

Código de Conduta do Tchelinux.

Inscrições

O evento tem entrada franca, porém os participantes são encorajados a doar 2kg de alimentos não perecíveis (exceto sal), que serão doados a instituições de caridade da região.

Os alimentos serão recebidos no momento do credenciamento.

As inscrições para o evento estarão abertas até o dia 29 de setembro de 2017, ou até se esgotarem as 180 vagas.

Inscreva-se agora!

Certificados

Serão fornecidos certificados digitais para os participantes do evento. Para obtê-los, você deverá utilizar o email fornecido na sua inscrição para o evento.

Leandro Lima: É funcionário público de manutenção. Inciou suas atividades em linux em 2015 ajudando a instalar e montar servidores linux (Biblivre 4) para Bibliotecas. Ajudou na elaboração e configuração do XUbuntu 16.04 para máquinas multiterminais.

Git para Desenvolvedores

O Git é uma ferramenta para control e gerenciamento de código fonte, inicialmente projetado e desenvolvido por Linus Torvalds para o desenvolvimento do kernel Linux, porém adotado por muitos outros projetos posteriormente devido a suas características. Nesta palestra veremos uma introdução ao Git e seus principais casos de uso dentro do fluxo de trabalho de um desenvolvedor.

Leonardo Vaz teve seu primeiro contato com Linux em 1998 e vem rodando ele em todos seus computadores desde então. Microsoft powered by Open-Source

Atualmente cerca de 40% de todas as aplicações em PaaS e VMs em IaaS operando no Microsoft Azure rodam algum software livre ou open-source. A estratégia da gigante de Redmond é se guinar completamente como provedora de serviços ao invés de apenas fornecedora de software. Esta palestra aborda as possibilidades e principais diferenciais do Azure como nuvem pública para hospedar soluções movidas à software livre.
